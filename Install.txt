How to install Julia.

0. Checkout official site for detailed instructions.
https://julialang.org/downloads/platform/

1. To install in ubuntu, try following.
sudo apt install julia



2. How to run Julia?

julia <ProgramName>
eg., julia helloWorld.jl

3. Other Resources to learn Julia
https://docs.julialang.org/en/v1/manual/getting-started/ (Official site)
https://www.coursera.org/learn/julia-programming/home/welcome
